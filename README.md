# Movie Booking App

A simple **React + Vite** movie booking app (frontend only). Uses **localStorage** for login, bookings, and admin features. No backend required.

## How to Run (Windows 11)

1. Install Node.js LTS  
   → Go to https://nodejs.org → Click the big green **LTS** button → Download & install the `.msi` file

2. Open terminal in the project folder:
   ```bash
   npm install
   ```

3. Start the application
    ```bash
    npm run dev
    ```
